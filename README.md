# Build Whatever

A retro-styled coding challenge generator

## Features

- Retro terminal aesthetic with pixel art and CRT effects
- Three difficulty levels: Easy, Medium, Hard
- Random design challenge generator

## How to Use

1. Open `index.html` in your web browser
2. Choose a difficulty level
3. Click "Load Challenge" until you find a prompt you like
4. Complete your design challenge!

## Technologies Used

- HTML5
- CSS3 (with modern features like clip-path for pixel art)
- JavaScript (Vanilla)
- Google Fonts (VT323 for the retro terminal look)

## Build Challenges

The app generates random build challenges with:
- A build type (e.g., "A SINGLE DETAIL VIEW")
